This is 'MDTC - Naitou', a simple Knight's Tour game celebrating Lunar New Year 2026 - Year of the Horse

## Building

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Rules

The game is based on the problem - Knight's Tour.

In a 8x8 chessboard, you will use a Knight to pass every cell, without return to a visited one.

The game is over when you successfully cover 64 cells, or when you unable to continue.
